# 🚀Guia de sobrevivência: Domínio do Terminal (CLI)

Este repósitorio foi criado como parte da disciplina de **Sistemas Operacionais**,
 utilizando o simulador 
 [Terminal Master] (http://andesonrmgomes.github.io/terminal-master/) 
 para praticar comandos essenciais de Windows e Linux.

---

 ## 📖Introdução
 A interface de Lnha de comando (CLI) é uma
 ferramenta pederosa que permite interagir com o 
 núcleo do sistema operacional. Dominar esses 
 comendos é fundamental para automação de tarefas,
 administração de servidores e desenvolvimento de 
 software.

---

 ## 💻 Comandos Aprendidos

 Abaixo, apresento a lista dos principais comandos 
 durante o treinamento gamificado:

|Comando |Função |Sistemas |Exemplo de uso|
|:--- | :--- | :--- |:---|
|´pvd´ | ´Exibe o diretório atual (caminho)` | ´Linux´ | `pvd´ |
| ´cd´ | ´Altera o diretório de trabalho´ | ´Ambos´ | ´cd/ home / 
usuario/documento´ |
| ´ls´ | ´Lista arquivos e pastas no diretório atual.´ | ´Unix/Linux´ | ´Lista arquivos e pastas de forma simples.´|
| ´dir´ | ´Listar o conteúdo de um diretório ou pasta no prompt de comando (CMD) do Windows, exibindo arquivos e subpastas.´ | ´Windows´ | ´Lista os arquivos e pastas no diretório atual. dir /w´|
| ´mkdir´ | ´Cria uma nova pasta no diretório atual.´ | ´Linux, macOS e Prompt de Comando do Windows´ | ´Criar novas pastas (diretórios). O uso básico é mkdir [nome_da_pasta].´|
| ´touch´ | ´Permite que você encoste nela e rapidamente o aparelho responde ao comando que você indicou´ | ´Os principais sistemas incluem telas sensíveis ao toque (capacitivas/resistivas) e softwares corporativos ´ | ´touch arquivo.txt´|
| ´type´ | ´Permitindo definir estruturas de objetos, tipos primitivos, uniões e interseções de forma flexível´ | ´Sistema de definição de tipos flexível, usado para criar aliases (apelidos) para tipos primitivos, objetos, uniões, interseções e tuplas´ | ´Usado para criar um nome personalizado para um tipo, facilitando a reutilização, organização e legibilidade do código´|
| ´rm´ | ´ Sua principal função é capturar, validar e processar os dados dos terminais (coletores) para otimizar o tratamento de horários, turnos e jornadas´ |  ´Sistema completo de gestão de Recursos Humanos, que atende aos processos como folha de pagamento, ponto eletrônico, cargos e salários, entre outros` | ´rm arquivo.txt´ |
| ´mv´ | ´Serve fundamentalmente para mover arquivos e diretórios de um local para outro e renomear arquivos ou pastas´ |  ´Soul MV` | ´mv arquivo_antigo.txt arquivo_novo.txt´ |
| ´cp´ | ´Copiar arquivos e diretórios de um local para outro´ |  ´Essencial para documentar o domínio do sistema ao criar backups, duplicar arquivos de configuração ou organizar documentos´ |  ´cp arquivo.txt arquivo_backup.txt` |
